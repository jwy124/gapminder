library(readr); library(dplyr); library(ggplot2)
set.seed(42)  # jitter 재현성

# document/ 또는 프로젝트 루트 어디서 렌더해도 동작
infile <- if (file.exists("data/gapminder.csv")) "data/gapminder.csv" else "../data/gapminder.csv"
gap <- read_csv(infile, show_col_types = FALSE)
gap$loggdp <- log(gap$gdpPercap)
FIRST <- min(gap$year); LAST <- max(gap$year)

theme_set(theme_minimal(base_size = 12))
cont_colors <- c(Africa="#E69F00", Americas="#56B4E9", Asia="#009E73",
                 Europe="#CC79A7", Oceania="#0072B2")
skew <- function(x){x<-x[!is.na(x)];m<-mean(x);mean((x-m)^3)/(mean((x-m)^2))^1.5}
kurt <- function(x){x<-x[!is.na(x)];m<-mean(x);mean((x-m)^4)/(mean((x-m)^2))^2-3}
노트

이 보고서는 data/gapminder.csv(142개국 × 5개 대륙 × 12개 시점, 1952–2007)를 분석한다. 모든 차트·표는 렌더 시점에 코드로 직접 재생성되므로 데이터가 바뀌면 보고서도 자동 갱신된다. 별도 명시가 없으면 통계는 비가중 국가 평균이며, 5장에서 인구가중과 비교한다.

5 비가중 vs 인구가중

국가당 1표(비가중)는 중국(13억)과 아이슬란드(30만)를 동급으로 본다. 인구가중은 “평균적인 사람”의 경험을 반영한다.

핵심 발견

  1. 분포 — gdpPercap·pop은 강한 우편향 → 로그·중앙값 필수. ’이봉형’은 풀링 착시였다(3장).
  2. 인구가중 ≠ 비가중 — ’평균적 사람’의 수명이 2007년 ’평균적 국가’를 추월(부호 역전). 가중 선택이 결론을 바꾼다(5장).
  3. 수렴 — 기대수명은 σ·β 모두 수렴, 소득은 σ-발산(6장).
  4. 관계 — lifeExp ~ log(GDP) R² ≈ 0.65, 로그형 수확체감. 강도는 1987년 정점 후 약화(7장).
  5. 잔차 — 부유하지만 단명한 에이즈·자원국 vs 소득 대비 장수국이 분리. 소득은 충분조건이 아니다.
  6. 후퇴국 — 소수(스와질란드·짐바브웨)지만 존재. 성장은 자동 보장이 아니다.

```{r setup}
# document/ 또는 프로젝트 루트 어디서 렌더해도 동작
infile <- if (file.exists("data/gapminder.csv")) "data/gapminder.csv" else "../data/gapminder.csv"
gap <- read_csv(infile, show_col_types = FALSE)
gap$loggdp <- log(gap$gdpPercap)
FIRST <- min(gap$year); LAST <- max(gap$year)

theme_set(theme_minimal(base_size = 12))
cont_colors <- c(Africa="#E69F00", Americas="#56B4E9", Asia="#009E73",
                 Europe="#CC79A7", Oceania="#0072B2")
skew <- function(x){x<-x[!is.na(x)];m<-mean(x);mean((x-m)^3)/(mean((x-m)^2))^1.5}
kurt <- function(x){x<-x[!is.na(x)];m<-mean(x);mean((x-m)^4)/(mean((x-m)^2))^2-3}
```

::: {.callout-note}
이 보고서는 `data/gapminder.csv`(142개국 × 5개 대륙 × 12개 시점, 1952–2007)를 분석한다.
모든 차트·표는 렌더 시점에 **코드로 직접 재생성**되므로 데이터가 바뀌면 보고서도 자동 갱신된다.
별도 명시가 없으면 통계는 **비가중 국가 평균**이며, [5장](#sec-weighted)에서 인구가중과 비교한다.
:::

# 비가중 vs 인구가중 {#sec-weighted}

국가당 1표(비가중)는 중국(13억)과 아이슬란드(30만)를 동급으로 본다. **인구가중**은 "평균적인 사람"의 경험을 반영한다.

```{r weighted}
glob <- gap |> group_by(year) |>
  summarise(`비가중` = mean(lifeExp), `인구가중` = weighted.mean(lifeExp, pop),
            .groups = "drop")
glob |> mutate(`차이` = round(`인구가중` - `비가중`, 1),
               across(c(`비가중`,`인구가중`), \(x) round(x, 1)))
```

```{r weighted-plot}
#| label: fig-weighted
#| fig-cap: "전지구 기대수명: 비가중(국가 평균) vs 인구가중(사람 평균)"
glob_long <- bind_rows(
  transmute(glob, year, value = `비가중`, 기준 = "비가중(국가 평균)"),
  transmute(glob, year, value = `인구가중`, 기준 = "인구가중(사람 평균)"))
ggplot(glob_long, aes(year, value, color = 기준)) +
  geom_line(linewidth = 1.2) + geom_point(size = 1.8) +
  scale_color_manual(values = c("비가중(국가 평균)"="#999999",
                                "인구가중(사람 평균)"="#D55E00")) +
  labs(x = "연도", y = "기대수명(세)", color = NULL)
```

::: {.callout-tip}
## 발견 — 부호 역전
1952년에는 인구가중 < 비가중(−0.1세)으로 인구 대국이 평균 이하였다. 그러나 2007년에는 인구가중 > 비가중(+1.9세)으로 **중국·인도가 세계 평균을 추월**했다. 가중 방식의 선택이 결론을 바꾼다.
:::

# 수렴의 정량화 {#sec-converge}

## σ-수렴 — 국가 간 분산이 줄어드는가

```{r sigma}
sigma <- gap |> group_by(year) |>
  summarise(`기대수명 CV` = sd(lifeExp)/mean(lifeExp),
            `log(GDP) CV` = sd(loggdp)/mean(loggdp), .groups = "drop")
sigma |> mutate(across(-year, \(x) round(x, 4)))
```

```{r sigma-plot}
#| label: fig-sigma
#| fig-cap: "σ-수렴: 변동계수(sd/mean) 추이. 값이 줄면 수렴."
sig_long <- bind_rows(
  transmute(sigma, year, cv = `기대수명 CV`, 지표 = "기대수명"),
  transmute(sigma, year, cv = `log(GDP) CV`, 지표 = "log(1인당 GDP)"))
ggplot(sig_long, aes(year, cv, color = 지표)) +
  geom_line(linewidth = 1.2) + geom_point(size = 1.8) +
  labs(x = "연도", y = "변동계수 (sd/mean)", color = NULL)
```

기대수명 CV는 0.249→0.180으로 **수렴**, log(GDP) CV는 0.137→0.157로 **발산**한다. 수명 격차는 좁혀지고 소득 격차는 벌어진다.

## β-수렴 — 가난했던 나라가 더 빨리 따라잡는가

```{r beta}
beta <- inner_join(
  filter(gap, year==FIRST) |> select(country, continent, le0 = lifeExp),
  filter(gap, year==LAST)  |> select(country, le1 = lifeExp), by = "country") |>
  mutate(growth = le1 - le0)
bfit <- lm(growth ~ le0, data = beta)
cat(sprintf("회귀: growth = %.1f + (%.3f)·le0,  기울기 p = %.2e  → %s\n",
            coef(bfit)[1], coef(bfit)[2], summary(bfit)$coefficients[2,4],
            ifelse(coef(bfit)[2] < 0, "음의 기울기 = β-수렴 성립", "발산")))
```

```{r beta-plot}
#| label: fig-beta
#| fig-cap: "β-수렴: 초기 기대수명(1952) vs 이후 증가폭(→2007)"
ggplot(beta, aes(le0, growth, color = continent)) +
  geom_point(alpha = 0.8, size = 2) +
  geom_smooth(method = "lm", formula = y~x, color = "black", se = TRUE) +
  scale_color_manual(values = cont_colors) +
  labs(x = "1952 기대수명(세)", y = "1952→2007 증가폭(세)", color = "대륙")
```

기울기가 유의하게 음수 → **출발점이 낮았던 국가일수록 더 크게 따라잡았다.**

# 기대수명 vs 소득 — 모형과 잔차 {#sec-model}

```{r model}
fit <- lm(lifeExp ~ loggdp, data = gap)
cat(sprintf("lifeExp ~ log(GDP):  R² = %.3f,  기울기 = %.2f세 / log단위\n",
            summary(fit)$r.squared, coef(fit)[2]))
```

소득이 로그 1단위(약 2.7배) 늘 때 기대수명이 평균 8.4세 증가 — 수확체감.

## 소득 대비 잔차 — 이상국가 식별

회귀선에서 가장 벗어난 국가는 소득만으로 설명되지 않는다.

```{r residuals}

over  <- g07r |> slice_max(resid, n = 5) |>
  transmute(country, continent, gdpPercap = round(gdpPercap), lifeExp, resid = round(resid,1))
under <- g07r |> slice_min(resid, n = 5) |>
  transmute(country, continent, gdpPercap = round(gdpPercap), lifeExp, resid = round(resid,1))
knitr::kable(over,  caption = "소득 대비 장수 (양의 잔차)")
knitr::kable(under, caption = "부유하지만 단명 (음의 잔차 — 에이즈·자원경제)")
```

```{r residual-plot}
ggplot(g07r, aes(gdpPercap, lifeExp)) +
```

음의 잔차 그룹은 **남부 아프리카 에이즈 유행국**(스와질란드·보츠와나·남아공)과 **자원경제**(적도기니·앙골라)다. 소득은 강력한 동인이지만 충분조건은 아니다.

# 핵심 발견 {.unnumbered}

1. **분포** — `gdpPercap`·`pop`은 강한 우편향 → 로그·중앙값 필수. '이봉형'은 풀링 착시였다([3장](#sec-shift)).
2. **인구가중 ≠ 비가중** — '평균적 사람'의 수명이 2007년 '평균적 국가'를 추월(부호 역전). 가중 선택이 결론을 바꾼다([5장](#sec-weighted)).
3. **수렴** — 기대수명은 σ·β 모두 수렴, 소득은 σ-발산([6장](#sec-converge)).
4. **관계** — `lifeExp ~ log(GDP)` R² ≈ 0.65, 로그형 수확체감. 강도는 1987년 정점 후 약화([7장](#sec-model)).
5. **잔차** — 부유하지만 단명한 에이즈·자원국 vs 소득 대비 장수국이 분리. 소득은 충분조건이 아니다.
6. **후퇴국** — 소수(스와질란드·짐바브웨)지만 존재. 성장은 자동 보장이 아니다.
